01 — Basics

DOM Kya Hota Hai?

Browser jab HTML page load karta hai, toh ek tree structure banata hai — isko DOM (Document Object Model) kehte hain. JavaScript se hum is tree ke kisi bhi node ko pakad sakte hain, badal sakte hain, add ya delete kar sakte hain.

01

document object

Poore page ka entry point. Sab kuch iske andar hota hai. document.body, document.title etc.

02

Element Select Karna

getElementById, querySelector, querySelectorAll se kisi bhi HTML element ko pakdo.

03

Element Banana & Hatana

createElement se naye elements banao, appendChild se lagao, remove() se hatao.

04

Events

addEventListener se user ke actions (click, keypress, scroll) pe react karo.
